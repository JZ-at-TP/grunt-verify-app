# grunt-verify-app

> Verifies an application listens on a port when spawned

```shell
npm install --save-dev grunt-verify-app
```

## Configuration

Here are the defaults

```js
verify_app: {
    server: {
        options: {
            port: 3000,
        }
    }
}
```
Run it!

```shell
grunt verify_app:server
```
